Estou desenvolvendo uma section no meu website fintech moderno nível Revoult, Stripe ou Apple. A intenção inicial era simplesmente adicionar o sticky para criar um efeito visual de sobreposição nos cards (.card_equipe) e tudo funcionou normalmente. Depois decidi adicionar um pouco mais de estilo visual, adicionei translate-x para alguns cards e um código JS para dar adicionar uma classe e retornar o translate-x-0 dos cards. Mas, por algum motivo, essa mudança alterou o comportamento definido para o meu header (principalmente em breakpoints inferiores a md:) que era fixo, mas deixou de ser após a adicionar a o translate-x para alguns cards (nth-[1]). Como posso resolver esse problema? Você acha que há algo melhor que podemos fazer (somente se for nível Revoult, Stripe ou Apple) ao invés dessa animação que pretendo? 

HTML:
<section class="sm:px-25 px-10 py-40 pt-35 flex flex-col items-center" id="quemSomos"> 
    <div class="text-center flex flex-col items-center justify-center max-w-3xl">
        <!-- Mini-texto sobre nós -->
    </div>
    <div class="flex flex-col items-center w-full gap-4 md:gap-2 mt-5 relative">
        <div class="card_equipe">
            <span class="texto_card_equipe">Card 1</span>
        </div>
        <!-- Mais cards -->
     </div>
</section>

CSS (TailwindCSS):

}

.card_equipe.visto {
    @apply  nth-[1]:md:translate-x-0 nth-[2]:translate-x-0 nth-[3]:translate-x-0 nth-[5]:translate-x-0 nth-[6]:translate-x-0 nth-[2]:xl:translate-x-96 nth-[3]:xl:-translate-x-96 nth-[4]:xl:-translate-x-96 nth-[5]:xl:translate-x-96 nth-[2]:lg:translate-x-50 nth-[3]:lg:-translate-x-50 nth-[4]:lg:-translate-x-50 nth-[5]:lg:translate-x-50
}

JavaScript: 
<script>
const observador = new IntersectionObserver(entrar => {
    entrar.forEach(cardEquipe => {
        if(cardEquipe.isIntersecting)
            cardEquipe.target.classList.add("visto")
        else 
            cardEquipe.target.classList.remove("visto")
    })
}, {threshold: 0.7})
const cardEquipe = document.querySelectorAll(".card_equipe").forEach(cards=> observador.observe(cards))
